# MiniMax 海螺视频生成

OmniMaaS 目前已支持对 MiniMax 海螺（Hailuo）视频大模型的统一接入与调用。通过本章节提供的接口说明，您可以在同一套 API 下发起视频生成请求，调用海螺视频生成接口生成视频。{{ className: 'lead' }}

<Note>
  调用视频生成模型生成视频时，系统会返回一个任务 ID；用户需要轮询状态接口来获取具体视频链接。
</Note>

---

## 支持模型

<Properties>
  <Property name="MiniMax-Hailuo-02" type="model">
    Hailuo-02 支持最高 1080P、最长 10s 的超清视频端到端生成，可直接根据文本或图片提示输出 30fps 的流畅片段。
  </Property>
  <Property name="MiniMax-Hailuo-2.3" type="model">
    MiniMax-Hailuo-2.3 视频生成模型，旨在实现逼真的运动、改进的视觉一致性和高保真风格化。
  </Property>
  <Property name="MiniMax-Hailuo-2.3-Fast" type="model">
    MiniMax Hailuo-2.3-Fast 图像到视频模型（Standard，768p），高级快速图像到视频生成。
  </Property>
</Properties>

其他模型支持可联系客服。

## 文生视频

<Row>
  <Col>
    <Endpoint method="POST" path="https://api.omnimaas.com/v1/video/generations" />

    ### 请求头

    <Properties>
      <Property name="Authorization" type="string" required>
        Bearer Token，格式：`Bearer YOUR_API_KEY`
      </Property>
      <Property name="Content-Type" type="string" required>
        请求体格式，固定为 `application/json`
      </Property>
    </Properties>

    ### 请求参数

    <Properties>
      <Property name="model" type="string" required>
        `MiniMax-Hailuo-02`、`MiniMax-Hailuo-2.3`
      </Property>
      <Property name="prompt" type="string" required>
        文本提示词，生成视频的文本描述
      </Property>
      <Property name="duration" type="integer">
        视频时长参数（秒），默认 `6`
      </Property>
      <Property name="size" type="string">
        画面比例，默认 `16:9`
        <Options>
          <Option value="16:9" />
          <Option value="1:1" />
          <Option value="9:16" />
        </Options>
      </Property>
      <Property name="metadata" type="object">
        扩展参数（JSON 字符串格式）
      </Property>
    </Properties>
  </Col>
  <Col sticky>
    <CodeGroup tag="POST" label="/v1/video/generations">

```bash {{ title: 'cURL' }}
curl -X POST https://api.omnimaas.com/v1/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "MiniMax-Hailuo-02",
    "prompt": "一只可爱的小猫在草地上玩耍",
    "duration": 6,
    "size": "16:9"
  }'
```

```python {{ title: 'Python' }}
import requests

url = 'https://api.omnimaas.com/v1/video/generations'

headers = {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
}

data = {
    "model": "MiniMax-Hailuo-02",
    "prompt": "夕阳下的海滩，海浪轻轻拍打着沙滩"
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(f"任务ID: {result['id']}")
print(f"状态: {result['status']}")
```

```javascript {{ title: 'Node.js' }}
const axios = require('axios');

const url = 'https://api.omnimaas.com/v1/video/generations';

const headers = {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
};

const data = {
    model: 'MiniMax-Hailuo-02',
    prompt: '夕阳下的海滩，海浪轻轻拍打着沙滩'
};

axios.post(url, data, { headers })
    .then(response => {
        console.log('任务ID:', response.data.id);
        console.log('状态:', response.data.status);
    })
    .catch(error => {
        console.error('请求失败:', error.message);
    });
```

  -H "Authorization: Bearer YOUR_API_KEY" \

## 响应

<Row>
  <Col>
    ### 响应参数

      <Property name="id" type="string">

      </Property>
      <Property name="object" type="string">

      </Property>

      </Property>

      </Property>
      <Property name="status" type="string">
        任务状态（processing: 处理中）
      </Property>
      <Property name="progress" type="integer">
        生成进度百分比
      </Property>
    </Properties>
  </Col>
  <Col sticky>
    <CodeGroup title="响应示例">

```json {{ title: '响应' }}
{
  "id": "{video_id}",
  "object": "video",
  "model": "MiniMax-Hailuo-02",
  "created_at": 1640995200,
  "status": "processing",
  "progress": 0
}
```

## 查询视频

根据任务 ID 查询视频生成任务的状态和结果

<Row>
  <Col>
    <Endpoint method="GET" path="https://api.omnimaas.com/v1/video/generations/{task_id}" />

    ### 请求头

    <Properties>
      <Property name="Authorization" type="string" required>
        Bearer Token，格式：`Bearer YOUR_API_KEY`
      </Property>
    </Properties>

    ### 路径参数

    <Properties>
      <Property name="task_id" type="string" required>
        视频任务 ID
      </Property>
    </Properties>

    ### 响应参数

    <Properties>
      <Property name="id" type="string">
        视频任务 ID
      </Property>
      <Property name="object" type="string">
        对象类型，固定为 `video`
      </Property>
      <Property name="model" type="string">
        使用的模型名称
      </Property>
      <Property name="created_at" type="integer">
        创建时间戳
      </Property>
      <Property name="status" type="string">
        任务状态（succeeded: 成功）
      </Property>
      <Property name="progress" type="integer">
        生成进度百分比
      </Property>
      <Property name="url" type="string">
        视频下载链接（完成时）
      </Property>
    </Properties>
  </Col>
