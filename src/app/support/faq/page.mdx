# 常见问题

## 账户相关

### 如何创建 OmniMaaS 账户？

访问 omnimaas.com 并点击"注册"按钮，按照提示完成注册流程。

### 如何重置密码？

在登录页面点击"忘记密码"，输入您的手机号，我们会发送重置验证码。

## API 相关

### API 请求有速率限制吗？

是的，具体限制请参考您的套餐说明。

### 如何获取 API 密钥？

登录后，在"账户"-"API Keys"页面可以创建和管理您的 API 密钥。

## 计费相关

### 有免费套餐吗？

是的，新用户注册即可获得免费额度体验。

### 如何充值？

在账户设置中选择"充值"，选择适合您的金额并完成支付。
